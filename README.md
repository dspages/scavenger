# Scavenger

Turn-based tactics game built in Unity.

## Unity version

This repository targets **Unity 6000.3.11f1** (Unity 6). Use the same editor version (or a compatible patch in the same minor line) to avoid project upgrade prompts and mismatched assets.

## Opening the project

1. Install [Unity Hub](https://unity.com/download) and add editor **6000.3.11f1** (or install it when Hub prompts you).
2. In Unity Hub, click **Add project** / **Open** and choose this folder: the directory that contains `Assets`, `Packages`, and `ProjectSettings` (the repo root).
3. Open the project from the Hub; Unity will import assets and compile scripts on first load.

If you clone from Git, open the cloned folder the same way—do not open only `Assets` as the project root.
